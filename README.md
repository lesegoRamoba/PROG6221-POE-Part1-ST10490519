# 📌 Project Overview

This project is a Cybersecurity Awareness Chatbot developed in two parts:

- Part 1: Console-based chatbot application
- Part 2: Windows Forms GUI chatbot application

The chatbot educates users about cybersecurity topics such as password safety, phishing, privacy, and safe browsing.

---

# ▶️ How to Run

## Part 1 (Console Application)
1. Clone this repository
2. Open the solution file (.sln) in Visual Studio
3. Ensure `greeting.wav` is in the output directory (set "Copy if newer")
4. Build and run (F5)

## Part 2 (Windows Forms GUI)
1. Open the `ChatbotGUI` folder in Visual Studio
2. Open the `.sln` file inside the folder
3. Set `ChatbotGUI` as the startup project
4. Run the application (F5)

---

# ✨ Features

## Part 1 – Console Chatbot
- Voice greeting on startup (WAV file)
- ASCII art logo display
- Personalised interaction (user name input)
- Cybersecurity awareness responses:
  - Password safety
  - Phishing detection
  - Safe browsing
  - Privacy protection
- Input validation for empty or unknown queries
- Enhanced console UI with colours and formatting

## Part 2 – GUI Chatbot (WinForms)
- Modern Windows Forms interface
- Chat-style messaging window
- Real-time user interaction
- Memory-based conversation (remembers user input)
- Sentiment-aware responses
- Delegate-based response system
- Keyword recognition with randomised replies
